# Danh sách phần mềm yêu cầu (Software Requirements)

Do giới hạn dung lượng của kho lưu trữ GitHub, các bộ cài đặt phần mềm không được đính kèm trực tiếp tại đây. Để chạy và chỉnh sửa đồ án này, vui lòng tải và cài đặt các phần mềm theo các liên kết chính thức dưới đây:

1. **Visual Studio Community 2026 (hoặc 2022):**
   - Môi trường phát triển mã nguồn C# và ASP.NET MVC.
   - [Link tải chính thức từ Microsoft](https://visualstudio.microsoft.com/vs/community/)

2. **Microsoft SQL Server (Developer/Express Edition):**
   - Hệ quản trị cơ sở dữ liệu lưu trữ CookingShareDB.
   - [Link tải chính thức từ Microsoft](https://www.microsoft.com/en-us/sql-server/sql-server-downloads)

3. **SQL Server Management Studio (SSMS):**
   - Công cụ quản lý và chạy file Script Database.
   - [Link tải chính thức từ Microsoft](https://learn.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms)
